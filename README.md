# slot-car-race-manager

Slot Car Race manager, server, and racer database for DYI slot car racer track controlled by Arduino.

# Byte Commands

0000 0000 = RESERVED
0000 0001 = START RACE
0000 0010 = START COUNTDOWN
0000 0011 = FALSE START
0000 0100
